한울학교 진로마트 한울점 최종본

실행 방법
1. ZIP 파일의 압축을 풉니다.
2. HanulDelivery_Header_Final 폴더의 index.html을 더블클릭합니다.
3. 관리자 기본 비밀번호는 1234입니다.

최종 헤더 구성
- 왼쪽: 한울학교 교표
- 가운데: 제공된 '진로마트 한울점 · 교내배달서비스' 배너 이미지
- 오른쪽: 음성 안내 버튼

주요 기능
- 6단계 주문
- 주문내역과 배달 상태 관리
- 전자서명
- 통계, 랭킹, 스탬프북
- 물품·교실·배달원 관리
- 백업·복원

데이터는 현재 브라우저의 LocalStorage에 저장됩니다.


[오류 수정]
주문 완료 후 홈에서 다시 주문하기를 누르면 주문 상태를 초기화하고 1단계 교실 선택 화면부터 시작하도록 수정함.


[v6 실시간 알림 추가]
- 관리자 새 주문 빨간 배지
- 확인할 때까지 4초 간격 반복 알림음(Web Audio)
- 교사용 실시간 주문 팝업
- 관리자 실시간 새 주문 목록 및 일괄 확인
- 배달완료 주문 카드 자동 초록색 표시
